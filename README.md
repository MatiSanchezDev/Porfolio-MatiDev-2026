## ⚙️ Desarrollo y despliegue

Sitio estático, sin build. **Subí todos estos archivos a la raíz del repo, respetando la estructura:**

```
índice/
├── index.html                 ← el portfolio
├── support.js                 ← runtime (necesario)
├── CV-Matias-Sanchez.html     ← CV online (botón "Ver CV online")
└── uploads/
    └── IMG_20250522_104016.jpg ← foto del hero
```

> Importante: mantené la carpeta `uploads/` y el archivo `support.js` junto al `index.html`, o la foto y el render no cargan.

### Desplegar en Vercel
1. Subí el repo a GitHub.
2. En [Vercel](https://vercel.com): **Add New → Project → Import** el repo.
3. Framework Preset: **Other**. Build Command y Output Directory: vacíos.
4. **Deploy**. Cada push a `main` redepliega automáticamente.

> Funciona igual en Netlify o GitHub Pages — es solo HTML estático.
